# sort_visualization
Nine sort algorithms visualization.八种排序可视化算法

由厦门大学本科生马承乾、李聪、金为轩、李亚轩合作完成

马承乾：设计框架、设计冒泡排序、归并排序、快速排序、基数排序

李聪：设计插入排序、堆排序、修改快速排序

金为轩：设计演示视频、演示PPT

李亚轩：设计希尔排序、选择排序
